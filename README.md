## 📋 درباره پروژه

این یک **لیست قیمت آنلاین** برای فروشگاه ورزشی اسپرت ۹۰ است که به مشتریان امکان مشاهده قیمت‌ها، جستجو و مقایسه محصولات را میدهد.

### ✨ قابلیت‌ها

- 🔍 **جستجوی پیشرفته** - جستجو در اسم و کد کالا
- 📂 **دسته‌بندی محصولات** - نمایش محصولات در دسته‌بندی‌های مختلف
- 🌓 **دارک/روشن مود** - تغییر تم با ذخیره‌سازی خودکار
- 📱 **واکنش‌گرا** - نمایش عالی در موبایل، تبلت و دسکتاپ
- 🏷️ **کد کالا** - هر محصول دارای کد یکتا
- 📊 **وضعیت موجودی** - نمایش موجود/ناموجود بودن محصولات
